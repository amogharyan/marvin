### Dubbing API Quickstart

Source: https://elevenlabs.io/docs/cookbooks/dubbing

This section outlines the steps to dub an audio file using the ElevenLabs Dubbing API. It includes setting up an API key, installing the necessary SDKs, and providing code examples in Python and TypeScript to perform the dubbing operation.

```APIDOC
## Create an API key

[Create an API key in the dashboard here](https://elevenlabs.io/app/settings/api-keys), which you’ll use to securely [access the API](/docs/api-reference/authentication).

Store the key as a managed secret and pass it to the SDKs either as a environment variable via an `.env` file, or directly in your app’s configuration depending on your preference.

```js title=".env"
ELEVENLABS_API_KEY=<your_api_key_here>
```

## Install the SDK

We'll also use the `dotenv` library to load our API key from an environment variable.

<CodeBlocks>
  ```python
  pip install elevenlabs
  pip install python-dotenv
  ```

  ```typescript
  npm install @elevenlabs/elevenlabs-js
  npm install dotenv
  ```
</CodeBlocks>

<Note>
  To play the audio through your speakers, you may be prompted to install [MPV](https://mpv.io/)
  and/or [ffmpeg](https://ffmpeg.org/).
</Note>

## Make the API request

Create a new file named `example.py` or `example.mts`, depending on your language of choice and add the following code:

<CodeBlocks>
  ```python maxLines=0
  # example.py
  import os
  from dotenv import load_dotenv
  from elevenlabs.client import ElevenLabs
  from elevenlabs.play import play
  import requests
  from io import BytesIO
  import time

  load_dotenv()

  elevenlabs = ElevenLabs(
    api_key=os.getenv("ELEVENLABS_API_KEY"),
  )

  target_lang = "es"  # Spanish

  audio_url =
      "https://storage.googleapis.com/eleven-public-cdn/audio/marketing/nicole.mp3"
  response = requests.get(audio_url)

  audio_data = BytesIO(response.content)
  audio_data.name = "audio.mp3"

  # Start dubbing
  dubbed = elevenlabs.dubbing.create(
      file=audio_data, target_lang=target_lang
  )

  while True:
      status = elevenlabs.dubbing.get(dubbed.dubbing_id).status
      if status == "dubbed":
          dubbed_file = elevenlabs.dubbing.audio.get(dubbed.dubbing_id, target_lang)
          play(dubbed_file)
          break
      else:
          print("Audio is still being dubbed...")
          time.sleep(5)
  ```

  ```typescript maxLines=0
  // example.mts
  import { ElevenLabsClient, play } from "@elevenlabs/elevenlabs-js";
  import "dotenv/config";

  const elevenlabs = new ElevenLabsClient();

  const targetLang = "es"; // spanish
  const sourceAudio = await fetch(
    "https://storage.googleapis.com/eleven-public-cdn/audio/marketing/nicole.mp3"
  );
  const audioBlob = new Blob([await sourceAudio.arrayBuffer()], {
    type: "audio/mp3",
  });

  // Start dubbing
  const dubbed = await elevenlabs.dubbing.create({
    file: audioBlob,
    targetLang: targetLang,
  });

  while (true) {
    const { status } = await elevenlabs.dubbing.get(
      dubbed.dubbingId
    );
    if (status === "dubbed") {
      const dubbedFile = await elevenlabs.dubbing.audio.get(
        dubbed.dubbingId,
        targetLang
      );
      await play(dubbedFile);
      break;
    } else {
      console.log("Audio is still being dubbed...");
    }

    // Wait 5 seconds between checks
    await new Promise((resolve) => setTimeout(resolve, 5000));
  }
  ```
</CodeBlocks>

### Quick Start: Initiate and Interact with an Agent Conversation in Swift

Source: https://elevenlabs.io/docs/agents-platform/libraries/swift

A quick start guide for initiating a conversation with an agent, observing its state and messages, and sending messages or controlling the conversation. Requires iOS 14.0+ / macOS 11.0+ and Swift 5.9+.

```swift
import ElevenLabs

// Start a conversation with your agent
let conversation = try await ElevenLabs.startConversation(
    agentId: "your-agent-id",
    userId: "your-end-user-id",
    config: ConversationConfig()
)

// Observe conversation state and messages
conversation.$state
    .sink { state in
        print("Connection state: (state)")
    }
    .store(in: &cancellables)

conversation.$messages
    .sink { messages in
        for message in messages {
            print("\(message.role): \(message.content)")
        }
    }
    .store(in: &cancellables)

try await conversation.sendMessage("Hello!")
try await conversation.toggleMute()
await conversation.endConversation()

	"fmt"
	"net/http"
	"io"
)

func main() {

	url := "https://api.elevenlabs.io/v1/audio-native/project_id/settings"

	req, _ := http.NewRequest("GET", url, nil)

	req.Header.Add("xi-api-key", "xi-api-key")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := io.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))

}

### Generate Detailed Music with Composition Plan

Source: https://elevenlabs.io/docs/cookbooks/music/quickstart

Generates music and provides detailed response including the composition plan and song metadata. This is useful for understanding how the music was generated or for reusing the composition plan. It requires a prompt and music length, and returns detailed track information including audio bytes.

```python

    prompt="Create an intense, fast-paced electronic track for a high-adrenaline video game scene. Use driving synth arpeggios, punchy drums, distorted bass, glitch effects, and aggressive rhythmic textures. The tempo should be fast, 130–150 bpm, with rising tension, quick transitions, and dynamic energy bursts.",
    music_length_ms=10000,
)

print(track_details.json) # json contains composition_plan and song_metadata. The composition plan will include lyrics (if applicable)
print(track_details.filename)
# track_details.audio contains the audio bytes
```

### POST /music/compose

Source: https://elevenlabs.io/docs/cookbooks/music/quickstart

Generates music either from a composition plan or a direct prompt. This endpoint is used to create the actual audio content.

```APIDOC
## POST /music/compose

### Description
Generates music either from a composition plan or a direct prompt. This endpoint is used to create the actual audio content.

### Method
POST

### Endpoint
/music/compose

### Parameters
#### Request Body
- **composition_plan** (object) - Optional - A pre-generated composition plan object.
- **prompt** (string) - Optional - The text prompt describing the desired music. Cannot be used with `composition_plan`.
- **music_length_ms** (integer) - Optional - The desired length of the music in milliseconds. Required if `prompt` is used and `composition_plan` is not.

### Request Example (using composition plan)
```json
{
  "composition_plan": {
    "positiveGlobalStyles": [
      "electronic",
      "fast-paced"
    ],
    "negativeGlobalStyles": ["acoustic", "slow"],
    "sections": [
      {
        "sectionName": "Intro",
        "positiveLocalStyles": ["rising synth arpeggio"],
        "negativeLocalStyles": ["soft pads"],
        "durationMs": 3000,
        "lines": []
      }
    ]
  }
}
```

### Request Example (using prompt)
```json
{
  "prompt": "An upbeat electronic track for a commercial.",
  "music_length_ms": 60000
}
```

### Response
#### Success Response (200)
- **audio_bytes** (binary) - The generated music as audio bytes.

#### Response Example
(Binary audio data)
```

### Generate Music with Eleven Labs API (Python)

Source: https://elevenlabs.io/docs/cookbooks/music/quickstart

This Python script demonstrates how to generate music using the Eleven Labs API. It initializes the client with an API key from environment variables, composes a music track based on a detailed prompt and duration, and then plays the generated track.

```python
# example.py
from elevenlabs.client import ElevenLabs
from elevenlabs.play import play
import os
from dotenv import load_dotenv
load_dotenv()

elevenlabs = ElevenLabs(
    api_key=os.getenv("ELEVENLABS_API_KEY"),
)

track = elevenlabs.music.compose(
    prompt="Create an intense, fast-paced electronic track for a high-adrenaline video game scene. Use driving synth arpeggios, punchy drums, distorted bass, glitch effects, and aggressive rhythmic textures. The tempo should be fast, 130–150 bpm, with rising tension, quick transitions, and dynamic energy bursts.",
    music_length_ms=10000,
)

play(track)
```
